# calibration_data/

Output directory for `scripts/run_calibration.py`. Each run writes a
Python module here that exposes:

  * `CALIBRATED_COSTS` — `{(tier, question): {"model": ..., "params": (a, b), "rms": ...}}`
  * `METADATA` — timestamp, machine info, bench version

The `default.py` file in this directory (when present) is the most
recent calibration committed for posterity. It is regenerated by
running the calibration script and committing the result.

User-specific calibrations should be named `<name>.local.py` so the
`.gitignore` doesn't accidentally commit them.
